├─────────────────────────────────────────────────────────────┤
│                                                             │
│  As a developer, I would like to implement an ad type       │
│  selection page that allows users to choose between three   │
│  ad formats:                                                │
│                                                             │
│    1. Text Ad — represented with a file-text icon and       │
│       light blue card background (#E6F2FF)                  │
│                                                             │
│    2. Image & Text Ad — represented with an image icon      │
│       and light green card background (#E6FFE6)             │
│                                                             │
│    3. Video Ad — represented with a play-circle icon        │
│       and light purple card background (#F0E6FF)            │
│                                                             │
│  Each ad type should be displayed as a clickable card       │
│  with a distinct color and icon to clearly differentiate    │
│  them. The cards should be built as reusable components     │
│  for consistency across the application.                    │
│                                                             │
│  Acceptance Criteria:                                       │
│  - Three ad type cards displayed horizontally               │
│  - Each card has an icon, label, and unique color           │
│  - Cards are reusable components                            │
│  - Cards have rounded corners (12px radius)                 │
