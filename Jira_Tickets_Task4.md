## Ticket #3: User Survey

- **Title:** Create User Experience Survey Page
- **Project:** Skyscanner
- **Reporter:** Mark Lo
- **Description:**
  As a developer, I would like to create a user survey page styled with Skyscanner branding. The page should include 5 questions with text input fields for user responses:
